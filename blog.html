Question-1: What are the different ways to select an element in the DOM?

Answer: We can select an element from DOM by following ways:

  1. .getElementById()
  2. .getElementsByName()
  3. .getElementsByTagName()
  4. .getElementsByClassName()
  5. .querySelector()
  6. .querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer: innerHTML contains HTML content with tags, innerText contains with only visible text content, textContent contains with all visible text content and hidden text.

Question-3: What is event delegation in the DOM?

Answer: Event delegation means add event listener to a parent element, not in every child element.

Question-4: What is event bubbling in the DOM?

Answer: Event bubbling in DOM means propagates an event from the target element up to its ancestors.

Question-5: How do you create, add, and remove elements using JavaScript?

Answer: We can create element by .createElement(), add element by .appendChild(), and remove element by .remove().
